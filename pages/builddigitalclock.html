How to Build a Digital Clock

When I was first planning this project, I was going to make a digital timer. As I got further into building it, I realized that before you learn how to walk, you should really learn how to crawl.

Dates and times in JavaScript are not as straight forward as you would believe them to be, so I decided to start at the beginning and create a digital clock. Of course, I decided to have some fun with it and spice it up by employing a few fun CSS tricks.

If you build this project, don't delete it because I plan on using it as the base for more projects in the future.

  • Create digital-clock-app-container div.
  • Create current-time-clock div inside of digital-clock-app-container.
  • Create current-date-clock div inside of digital-clock-app-container.
  • Create top-bezel-smartphone.app-open class and settings.
  • Create bottom-bezel-smartphone.app-open class and settings.
  • Create digital-clock-app-container class and settings
  • Create digital-clock-app-container.open class and settings
  • Create current-time-clock class and settings.
  • Create current-date-clock class and settings.
  • Create variable called smartPhoneContainer by using getElementById on the id smart-phone-container.
  • Create a click eventListener on smartPhoneContainer.
  • Pass in an event (e)
  • IF statement. If the e.target class list contains 'digital-clock-app,' run the following.
  • Run digitalClockApp( ) function. Will be created in next step.
  • Create variables for digitalClockAppDiv, topBezel, and bottomBezel.
  • Add the class of 'app-open' to topBezel and bottomBezel.
  • Add the class of 'open' to digitalClockAppDiv
  • setInterval on the digitalClockApp function to make it run once every second.
  • Create the digitalClockApp( ) function.
  • Create variable for currentTime by using getElementById on the id of 'current-time-clock.'
  • Create variable for currentDate by using getElementById on the id of 'current-date-clock.'
  • Create a variable called date that is linked to the JavaScript 'new Date;' function.
  • Create variables for hours, minutes, seconds, year, month, and date. Use the date variable (JavaScript function) we created in the last step for each one.
  • Add a +1 to month because month is zero indexed (eg. January is 0, February is 1, etc)
  • Create an IF/ELSE IF statement to convert from a 24 hour clock to a 12 hour clock.
  • IF hours are over 12, subtract 12 from the variable.
  • ELSE IF hours are equal to 0 (midnight to 1 AM), set hours to 12
  • The next 3 IF statements will add 0 before the variable if they are less than 10.
  • Use template literals to set the currentTime.innerHTML
  • Use template literals to set the currentDate.innerHTML
