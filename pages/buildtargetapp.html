When I was first planning this project, I was going to make a digital timer. As I got further into building it, I realized that before you learn how to walk, you should really learn how to crawl.

Dates and times in JavaScript are not as straight forward as you would believe them to be, so I decided to start at the beginning and create a digital clock. Of course, I decided to have some fun with it and spice it up by employing a few fun CSS tricks.

If you build this project, don't delete it because I plan on using it as the base for more projects in the future.

  • smartPhoneContainer variable will already be made from the digitalClockApp project.
  • Create a click eventListener on smartPhoneContainer.

  • Run weatherApp( ) function. Since it will throw an error when we push the app icon, comment it out for now until we create the function.
  • Create variables for weatherAppDiv, topBezel, and bottomBezel.

  • Add the class of 'open' to weatherAppDiv
  • Add IF/ELSE IF to the smartPhoneHomeButtonEventListener
  • IF appOpen is equal to false, meaning no app is open, we will continue with the current commands we had in here to open an app.
  • If an app is open (appOpen === true), we want pushing the home button to clear the screen and repopulate the DOM with our icons
  • Set appOpen to false to let the script know that the app is closed
  • Run the populateHomeScreen( ) function to start rebuilding the DOM.
  • Create variables for digitalClockAppDiv, topBezel, and bottomBezel
  • Add the class of 'slide-right' to digitalClockAppDiv. This will slide the div to the right 100%.
  • Remove the class of 'app-open' from topBezel and bottomBezel. This will remove the borders from the bezels.
  • Create a setTimeout function to remove the classes of 'open' and 'slide-right' from the digitalClockAppDiv. This will return it to it's default location. The reason you want to use a setTimeout here is so they don't disappear automatically when they are removed and the animation has time to play.
